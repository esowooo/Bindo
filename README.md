# Bindo App
This is a practice project built with Swift, UIKit, and CoreData.
Bindo allows you to create and manage subscription records. 
It also provides statistics and calendar functions to analyze subscription trends.
The App provides two different types of subscriptions: interval, date.
Interval allows you to set interval time where subscription is paid.
Date allows you to manually set payment dates.
The App supports 3 languages: English(main), Japanese, and Korean, depending on user's default language.
