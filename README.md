# Bindo App
## Features
- Subscription management (interval & date)
- CoreData persistence
- Multi-language support (EN / JP / KR)
- Statistics dashboard
- Calendar view

## Architecture
- MVC
- UIKit + CoreData

## How to Build
1. Clone repository
2. Open `Bindo.xcodeproj` in Xcode
3. Build & Run (iOS 15+)
